# Информация о проекте RentalAll

## Общие сведения

**Название проекта:** RentalAll - Система бронирования площадок для мероприятий

**Тип работы:** Дипломная работа

**Год разработки:** 2025

**Цель проекта:** Разработка информационной системы-агрегатора для бронирования площадок и аудиторий для мероприятий

## Обоснование актуальности

В современном мире проведение мероприятий различного масштаба - конференций, семинаров, тренингов, фотосессий - требует поиска подходящих площадок. Существующие решения часто:

- Ограничены по функционалу
- Не предоставляют удобный поиск и фильтрацию
- Не имеют системы онлайн-бронирования
- Не поддерживают систему отзывов и рейтингов

**RentalAll** решает эти проблемы, предоставляя:
- Единую платформу для поиска площадок
- Удобную систему бронирования и оплаты
- Систему отзывов и рейтингов
- Административную панель для управления

## Схема базы данных

### Основные таблицы:

1. **users_user** - Пользователи
   - Хранит информацию о пользователях
   - Роли: user, admin
   - Связь с bookings, reviews

2. **venues** - Площадки
   - Основная информация о площадках
   - Связь с images, categories, bookings, reviews

3. **venue_images** - Фотографии площадок
   - Множество изображений для одной площадки
   - Связь 1:N с venues

4. **categories** - Категории
   - Типы площадок
   - Связь M:N с venues

5. **venue_categories** - Связь площадок и категорий
   - Промежуточная таблица M:N

6. **bookings** - Бронирования
   - Информация о бронированиях
   - Статусы: pending, confirmed, cancelled
   - Связь с users, venues, payments

7. **payments** - Платежи
   - Информация об оплате
   - Статусы: pending, paid, failed
   - Связь с bookings

8. **reviews** - Отзывы
   - Отзывы пользователей
   - Рейтинг 1-5
   - Модерация администратором
